IPFS client served by GitHub Pages!

This page creates an IPFS node in your browser, exposed to your Javascript code as ipfsNode.

Note that opening two tabs of this page in the same browser won't work well. They will share node configuration. You'll end up trying to run two instances of the same node, with the same private key and identity, with unpredictable results.

Opening the same page in both FireFox and Chrome, will work!
